# Guided activation (browser)

**Activation spine:** first **quick** proof on your capture (preview-only) → optional **Formalize** with a draft registry when the path matters → **`agentskeptic check`** for decision-grade truth → **`agentskeptic enforce`** for repeatable CI gates.

The hosted page at **`/integrate/guided`** helps you produce local NDJSON + a suggested `quick` command; it does not replace running **`agentskeptic quick`** yourself.

**Normative integrator SSOT:** [`docs/integrate.md`](integrate.md) (quick preview boundary, contract check, and links to enforce semantics).
